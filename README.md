# HakedisCheck

Excel tabanli hak edis dogrulama araci.

## Icerik

- `HakedisCheck.Core`: Excel okuma, kolon profilleri, esitlestirme, agregasyon ve dogrulama mantigi
- `HakedisCheck.App`: WinForms arayuzu
- `HakedisCheck.Tests`: core davranis testleri

## Durum

- Core katmani derleniyor.
- Testler geciyor.
- WinForms proje yapisi hazir, ancak WindowsDesktop SDK gerektirdigi icin Windows ortaminda derlenmeli.
